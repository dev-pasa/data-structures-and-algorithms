# Binary Tree Height


## Challenge
* Find the height of the tree.
* Return and int that represents the height of the binary tree. 


## Approach & Efficiency
* Time Complexity = O(n): Must traverse the whole tree
* Space Compexity = O(1): Where h is the height of the tree
